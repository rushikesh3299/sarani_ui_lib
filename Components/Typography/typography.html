Typography

Typography can be called as text utilities. Checkout below text utilities for more information.

Examples of Typography

As shown in below code you can vary font sizes for headers & other ypography using 'fsxx' classes which are nothing but font size. e.g. fs05 -> font-size:0.5rem & fs35 -> font-size:3.5rem.
for font weight you can use 'fwxxx' classes which are from fs100 to fs900. E.g. fw200 -> font-weight:200 & fw900 -> font-weight:900
